# Expense Tracker

Description:
A full-stack Expense Tracker application to manage expenses, budgets, bills, savings, reports, grocery tracking, and payment methods.

Features:
- Authentication (Login/Register)
- Dashboard Analytics
- Monthly Budgets
- Expense Tracking
- Bills Management
- Grocery Module
- Savings Tracker
- Reports & Insights
- Payment Methods
- Notifications
- Dark Mode
- Responsive UI

Tech Stack:
Frontend:
- Angular
- TypeScript
- SCSS

Backend:
- Spring Boot
- Java
- Spring Security
- JWT Authentication
- JPA/Hibernate

Database:
- PostgreSQL

Setup Instructions:

Frontend:
cd frontend
npm install
ng serve

Backend:
cd expense-tracker
./mvnw spring-boot:run

Database:
Make sure PostgreSQL is running on your local machine and create a database as specified in your application properties.
